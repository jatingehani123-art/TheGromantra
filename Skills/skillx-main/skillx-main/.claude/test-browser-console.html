SkillX Leaderboard Debug Instructions

Manual Testing Steps:

  1. Open http://localhost:5173/ in a new tab
  2. Open Browser DevTools (F12 or Cmd+Option+I)
  3. Go to Console tab
  4. Look for any errors or warnings (especially hydration-related)
  5. Go to Elements/Inspector tab
  6. Find the leaderboard table (search for "tbody")
  7. Check if table rows exist in the DOM
  8. Check computed styles for visibility/display

Things to Check:

  • Console Errors: Any JavaScript errors?
  • Hydration Warnings: React hydration mismatch messages?
  • DOM Structure: Are the 30 table rows present?
  • CSS Visibility: Are rows hidden with display:none or visibility:hidden?
  • Network Tab: Are all resources loading successfully?

Run this in Browser Console:

// Check leaderboard state
const tbody = document.querySelector('tbody');
console.log('tbody exists:', !!tbody);
console.log('row count:', tbody?.querySelectorAll('tr').length);
console.log('tbody HTML (first 500 chars):', tbody?.innerHTML.substring(0, 500));

// Check visibility
const section = document.querySelector('div.mt-16.w-full.max-w-6xl.pb-16');
if (section) {
  const styles = window.getComputedStyle(section);
  console.log('Section visibility:', {
    display: styles.display,
    visibility: styles.visibility,
    opacity: styles.opacity
  });
}
